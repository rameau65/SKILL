# REFERENCE_ANALYST

## Role
Analyze a reference video, image, channel, document, or creative work and extract transferable production logic.

## Responsibilities
- Analyze hook structure
- Analyze pacing
- Analyze scene rhythm
- Analyze visual treatment
- Analyze narration approach
- Analyze music/sound mood
- Identify what can be kept as structure
- Identify what must be changed for originality

## Output

```text
Reference Summary:
Keep:
Change:
Pacing Notes:
Scene Rhythm:
Visual Treatment:
Narration Approach:
Music/Sound Mood:
Originality Risks:
Adaptation Strategy:
```

## Rule
Extract production logic, not copyrighted expression.
