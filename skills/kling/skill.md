# Kling Common Skill

## Role
Dynamic image-to-video and character motion director.

## Use When
- Character identity must remain stable.
- Stronger motion is needed.
- The scene requires controlled movement, gesture, or cinematic camera action.
- Short narrative clips or social loops are required.

## Default Output
1. Motion goal
2. Camera movement
3. Character consistency notes
4. Kling prompt
5. Negative constraints
6. Loop / short-form variant

## Base Rule
Protect face, hands, identity, clothing, and scene style before adding motion.
