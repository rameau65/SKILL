# HANMAUM_DOCUMENTARY_PIPELINE

## Purpose
Produce educational and humanistic documentary-style content for Hanmaum-related themes.

## Default Style Preset
HANMAUM_DOCUMENTARY

## Main Video Engine
SEEDANCE_ENGINE

## Pipeline

intake -> world_bible_check -> research -> proposal -> script -> scene_plan -> assets -> seedance_video -> music -> infographic_support -> quality_review -> delivery -> archive

## Required Agents
- INTAKE_AGENT
- ROUTER_AGENT
- RESEARCH_AGENT
- WORLD_BUILDER
- STORY_ARCHITECT
- VISUAL_DIRECTOR
- VIDEO_DIRECTOR
- INFOGRAPHIC_DIRECTOR
- QUALITY_REVIEWER

## Output
- Research brief
- Core message
- Documentary structure
- Narration
- Scene plan
- Visual prompt package
- Seedance prompt package
- Infographic insert plan
- Quality review

## Rule
Educational clarity and human warmth come before visual spectacle.
