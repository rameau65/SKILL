# CINEMATIC_WELLNESS_PIPELINE

## Purpose
Produce calm reflective cinematic films using the DEVO-AI-OS agentic workflow.

## Default Style Preset
MEDITATION_CINEMA

## Main Video Engine
SEEDANCE_ENGINE

## Pipeline

intake -> world_bible_check -> emotional_core -> script -> scene_plan -> visual_prompts -> seedance_prompts -> music_prompt -> quality_review -> delivery_package -> archive

## Required Agents
- INTAKE_AGENT
- ROUTER_AGENT
- WORLD_BUILDER
- STORY_ARCHITECT
- VISUAL_DIRECTOR
- VIDEO_DIRECTOR
- MUSIC_PRODUCER
- QUALITY_REVIEWER
- ARCHIVE_AGENT

## Output
- Core message
- Script
- Storyboard
- Image prompts
- Seedance prompts
- Music prompts
- Poster or thumbnail copy
- Quality review

## Rule
Silence, breath-like pacing, and visual space are part of the design.
