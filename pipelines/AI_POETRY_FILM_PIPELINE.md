# AI_POETRY_FILM_PIPELINE

## Purpose
Create symbolic AI art films where emotion becomes landscape and thought becomes cinematic metaphor.

## Default Style Preset
AI_POETRY_FILM

## Main Video Engine
SEEDANCE_ENGINE

## Pipeline

intake -> emotional_map -> visual_metaphor -> scene_plan -> image_prompts -> seedance_video -> ambient_music -> quality_review -> archive

## Required Agents
- INTAKE_AGENT
- ROUTER_AGENT
- WORLD_BUILDER
- STORY_ARCHITECT
- VISUAL_DIRECTOR
- VIDEO_DIRECTOR
- MUSIC_PRODUCER
- QUALITY_REVIEWER

## Output
- Emotional map
- Symbol list
- Scene plan
- Image prompts
- Seedance prompts
- Music prompt
- Quality review

## Rule
The film should feel like an inner journey, not a random surreal montage.
