# SHORTS_CINEMA_PIPELINE

## Purpose
Create 5-15 second vertical cinematic loops and YouTube Shorts.

## Default Style Preset
YOUTUBE_SHORTS_CINEMA

## Main Video Engine
SEEDANCE_ENGINE

## Supporting Engine
KLING_ENGINE for stronger motion.

## Pipeline

intake -> hook_design -> vertical_scene_plan -> image_prompt -> seedance_or_kling_video -> caption_copy -> quality_review -> upload_package

## Required Agents
- INTAKE_AGENT
- ROUTER_AGENT
- STORY_ARCHITECT
- VISUAL_DIRECTOR
- VIDEO_DIRECTOR
- CHANNEL_PRODUCER
- QUALITY_REVIEWER

## Output
- Hook
- 5-15 second scene plan
- Vertical image prompt
- Video prompt
- Caption copy
- Title variants
- Hashtag direction

## Rule
One short should contain one clear emotional hook.
