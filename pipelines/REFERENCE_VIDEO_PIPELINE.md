# REFERENCE_VIDEO_PIPELINE

## Purpose
Start from a video the user already loves and convert it into a grounded, differentiated production plan.

This pipeline does not copy the reference. It extracts useful production DNA:

- pacing
- hook style
- structure
- tone
- scene rhythm
- camera language
- visual treatment
- narration approach
- music/sound mood

Then it transforms those elements into a new original work.

## Input
- YouTube video / Short / Reel / TikTok / local clip
- Target topic
- Target duration
- Target audience
- Desired platform
- Available tools

## Pipeline Stages

```text
reference_intake
→ transcript_analysis
→ pacing_analysis
→ scene_keyframe_analysis
→ style_analysis
→ concept_variants
→ tool_path
→ cost/time estimate
→ sample scene
→ approval
→ full production
```

## Output

1. What to keep from the reference
2. What to change for originality
3. 2–3 differentiated concepts
4. Recommended tool path
5. Estimated production complexity
6. Sample storyboard / sample scene
7. Approval checklist

## Agent Roles

- INTAKE_AGENT: collects reference and target topic
- REFERENCE_ANALYST: analyzes structure, pacing, style, hook
- CONCEPT_DEVELOPER: creates differentiated variants
- TOOL_PATH_PLANNER: selects engines and tools
- CREATIVE_DIRECTOR: approves direction
- QUALITY_REVIEWER: prevents imitation and checks originality

## Rule
Keep structure, pacing, or energy only. Do not imitate protected characters, logos, proprietary visual identity, or distinctive copyrighted expression.
