# QUALITY_STANDARD

## Review Categories

| Category | Question |
|---|---|
| Clarity | 초등학생도 이해할 수 있는가? |
| Narrative | 이야기 흐름이 있는가? |
| Visuality | 이미지/도식으로 바꾸기 쉬운가? |
| Emotion | 감정선이 살아 있는가? |
| Tool Fit | 실제 AI 도구에 입력 가능한가? |
| Brand Fit | 프로젝트 톤과 맞는가? |
| Expandability | 다른 장르로 확장 가능한가? |

## Score

- A+ : 바로 제작 가능
- A : 약간의 보완 후 제작 가능
- B : 구조는 있으나 디테일 보완 필요
- C : 재구성 필요
- D : 목적 불명확

## Review Output

1. Score
2. Strengths
3. Problems
4. Revision Strategy
5. Improved Version
