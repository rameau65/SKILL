# OUTPUT_STANDARD

Every DEVO-AI-OS output should be reusable across tools and formats.

## Master Output Format

1. Core Message
2. One-line Summary
3. Audience
4. Story Structure
5. Visual Strategy
6. Video/Motion Strategy
7. Music/Sound Strategy
8. Infographic/Document Structure
9. Distribution Plan
10. Quality Checklist

## Prompt Output Rule

When creating prompts:

- Korean purpose first
- English generation prompt second
- Parameters last
- Negative constraints included when needed

## Example

```text
Purpose: 명상 영상용 새벽 골목 장면
Prompt: cinematic Korean dawn alley, soft blue hour light...
Parameters: --ar 16:9 --stylize 250 --v 8.1
Avoid: text artifacts, distorted hands, extra limbs
```
