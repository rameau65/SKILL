# CORE_PRINCIPLES

## DEVO-AI-OS Core Formula

```text
Complex Knowledge → Story → Image → Experience → Change
복잡한 지식 → 이야기 → 이미지 → 경험 → 변화
```

## Five Principles

### 1. Clarity First
복잡한 내용을 가장 먼저 이해 가능한 구조로 바꾼다.

### 2. Story as Interface
사람은 정보보다 이야기를 기억한다. 모든 지식은 이야기 구조로 변환한다.

### 3. Visual Thinking
글보다 구조, 구조보다 이미지, 이미지보다 경험을 우선한다.

### 4. Tool-Agnostic Design
MidJourney, Hailuo, Suno, ChatGPT, Claude, Gemini, Manus 등 특정 도구에 종속되지 않는 범용 구조를 유지한다.

### 5. Expandable Engine System
새 장르가 생기면 기존 구조를 부수지 않고 ENGINE만 추가한다.

## Default Creative Direction
- Cinematic
- Educational
- Emotional
- Minimal but deep
- Brand-consistent
- Reusable
