# AGENTIC_OPERATING_SYSTEM

## Version
DEVO-AI-OS v7.0

## Purpose
Evolve DEVO-AI-OS from an engine-based creative system into an agentic creative operating system.

The system no longer asks only, "Which engine should generate this?" It asks:

```text
What pipeline is this?
Which agents must participate?
What must be researched, proposed, approved, produced, reviewed, and archived?
```

## Core Influence
Inspired by agentic video production architecture patterns such as:

- reference-driven production planning
- pipeline-first task selection
- stage director skills
- self-review gates
- state checkpoints
- human approval at creative decision points
- auditable decision logs

## Agentic Flow

```text
User Request
→ INTAKE_AGENT
→ ROUTER_AGENT
→ RESEARCH_AGENT
→ PROPOSAL_AGENT
→ CREATIVE_DIRECTOR
→ STAGE_AGENTS
→ QUALITY_REVIEWER
→ DELIVERY_AGENT
→ ARCHIVE_AGENT
```

## Universal Production Stages

```text
intake
→ reference_analysis
→ research
→ proposal
→ script
→ scene_plan
→ assets
→ video
→ music
→ edit_package
→ quality_review
→ delivery
→ archive
```

## Rule
Every serious creative request should be treated as a pipeline selection problem first, then an engine execution problem.
