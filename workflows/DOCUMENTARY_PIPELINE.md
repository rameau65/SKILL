# DOCUMENTARY_PIPELINE

## Flow

Subject
→ Core Question
→ Interview Questions
→ Narrative Arc
→ Scene List
→ Archive Material Plan
→ Narration
→ Visual Style
→ Music Direction
→ Edit Structure

## Output
- Documentary concept
- 3-act structure
- Interview guide
- Scene board
- Narration draft
- Visual prompt package
- Music prompt package

## Rule
A documentary is driven by a question, not by information alone.
