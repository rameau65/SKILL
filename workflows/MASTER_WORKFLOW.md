# MASTER_WORKFLOW

## Purpose
A universal pipeline for turning one topic into a complete creative package.

## Flow

```text
Topic
→ Core Message
→ Story Structure
→ World Bible Check
→ Visual Direction
→ Video Direction
→ Music Direction
→ Infographic/Document Package
→ Channel/Distribution Package
→ Quality Review
→ Archive
```

## Default Deliverables
1. Summary
2. Storyboard
3. Image prompts
4. Video prompts
5. Music prompts
6. Infographic/PPT structure
7. YouTube package
8. Review report

## Rule
The workflow should produce both creative output and reusable system assets.
