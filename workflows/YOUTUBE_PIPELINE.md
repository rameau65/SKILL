# YOUTUBE_PIPELINE

## Flow

Channel Concept
→ Content Pillars
→ Episode Idea
→ Script
→ Visual Package
→ Video Package
→ Music Package
→ Thumbnail
→ Title
→ Description
→ Shorts Repurpose
→ Archive

## Output Package
- Long-form script
- 3 Shorts scripts
- Thumbnail prompt
- Title variants
- Description
- Tags
- Community post

## Rule
Every long-form video should produce multiple short-form and archive assets.
