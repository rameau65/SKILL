# MEDITATION_PIPELINE

## Flow

Theme / Emotion
→ Meditation Message
→ 1/3/5 Minute Script
→ Visual Key Scenes
→ MidJourney Prompts
→ Hailuo/Grok Prompts
→ Suno Music Prompt
→ Poster / Thumbnail Copy
→ Quality Review

## Default Durations
- 1 minute: single insight
- 3 minutes: guided emotional shift
- 5 minutes: full breathing and reflection
- 10 minutes: deep meditation session

## Rule
Keep movement, music, and narration spacious. Silence is part of the design.
