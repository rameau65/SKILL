# BOOK_PIPELINE

## Flow

Idea
→ Book Positioning
→ Table of Contents
→ Chapter Outline
→ Sample Chapter
→ Full Draft Plan
→ Cover Direction
→ Marketing Copy
→ Publishing Package

## Output
- Book concept
- Reader profile
- TOC
- Chapter summaries
- Writing schedule
- Cover prompt
- Back cover copy
- Launch content

## Rule
A book project must be both readable and packageable.
