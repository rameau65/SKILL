# INSTALL

DEVO-AI-OS is a Markdown-based creative operating system. It does not require installation in the traditional software sense.

## Use with ChatGPT

1. Create a new Project.
2. Add `core/SYSTEM_PROMPT.md` as project instructions.
3. Upload relevant engine files from `engines/`.
4. Add project-specific files from `world_bible/`.
5. Use templates from `templates/`.

## Use with Claude

1. Create a Skill folder.
2. Put `core/SYSTEM_PROMPT.md` as the main instruction file.
3. Add engine markdown files as references.
4. Add project-specific World Bible files.

## Use with Gemini / Manus

1. Add the repository documents as a knowledge base.
2. Set `core/SYSTEM_PROMPT.md` as the master behavior guide.
3. Use workflows as reusable execution plans.

## Recommended Minimal Set

- README.md
- core/SYSTEM_PROMPT.md
- core/OUTPUT_STANDARD.md
- engines/STORY_ENGINE.md
- engines/VISUAL_ENGINE.md
- engines/VIDEO_ENGINE.md
- engines/MUSIC_ENGINE.md
- engines/QUALITY_ENGINE.md
- workflows/MASTER_WORKFLOW.md
