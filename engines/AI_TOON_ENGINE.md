# AI_TOON_ENGINE

## Purpose
Create character-based comics, webtoons, educational cartoons, and short visual stories.

## Input
- Theme
- Character
- Lesson
- Tone
- Target platform

## Output
- 4-cut comic
- Instagram toon
- Webtoon episode outline
- Educational comic
- Character sheet
- Scene prompt package

## Structure
Setup → Contrast → Realization → Punch/Insight

## Rule
For educational or philosophical topics, use humor, character reaction, and simple visual metaphor.
