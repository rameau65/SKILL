# VISUAL_ENGINE

## Purpose
Convert story, concept, or world bible elements into image-generation prompts.

## Tools
MidJourney, Niji, Flux, Imagen, Stable Diffusion.

## Output
1. Master prompt
2. Shot-by-shot prompts
3. Poster prompt
4. Thumbnail prompt
5. Style notes
6. Negative constraints

## Prompt Formula
Subject + Action + Environment + Lighting + Camera + Style + Mood + Detail + Parameters

## Default Parameters

### Cinematic
`--ar 16:9 --stylize 250 --v 8.1`

### Animation / Niji
`--ar 16:9 --stylize 300 --niji 7`

### Poster / Infographic
`--ar 2:3 --stylize 150 --v 8.1`

## Rule
Advertising quality > storytelling > detail > realism.
