# SCRIPT_ENGINE

## Purpose
Adapt source material into scripts for YouTube, documentary, shorts, lecture, meditation, or narration.

## Input
- Essay
- Novel
- Lecture
- Interview
- Philosophy document
- Report

## Output
- YouTube script
- Documentary script
- Shorts script
- Narration script
- Lecture script
- Scene-by-scene adaptation

## Structure
Hook → Context → Main Development → Turning Point → Insight → Closing

## Rule
A script must be spoken, seen, and timed. Avoid purely written-language density.
