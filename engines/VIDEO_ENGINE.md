# VIDEO_ENGINE

## Purpose
Route storyboards, images, scripts, and concepts into the correct independent video-generation engine.

VIDEO_ENGINE is now a router and supervisor, not a single prompt format.

## Independent Video Engines

```text
SEEDANCE_ENGINE      cinematic sequence / emotional continuity
KLING_ENGINE         character motion / dynamic image-to-video
HAILUO_ENGINE        subtle image animation / gentle camera motion
GROK_VIDEO_ENGINE    fast experimental video / native audio tests
RUNWAY_ENGINE        edit-friendly video generation and refinement
SORA_ENGINE          high-level cinematic/world simulation planning
FLOW_ENGINE          production orchestration across all tools
```

## Input
- Storyboard
- Reference images
- Scene goal
- Character consistency needs
- Motion intensity
- Duration
- Target platform
- Music / narration needs

## Engine Selection Guide

| Need | Recommended Engine |
|---|---|
| Cinematic emotional sequence | SEEDANCE_ENGINE |
| Character movement / dynamic shot | KLING_ENGINE |
| Gentle image animation | HAILUO_ENGINE |
| Quick experiment / audio mood | GROK_VIDEO_ENGINE |
| Production workflow | FLOW_ENGINE |
| Long-term high-end cinematic planning | SORA_ENGINE |

## Output Table

| Cut | Best Engine | Length | Image/Scene | Camera | Subject Motion | Atmosphere | Prompt |
|---|---|---:|---|---|---|---|---|

## Universal Image-to-Video Base Prompt
Animate this still image into a cinematic shot while preserving the original composition, identity, color palette, and style. Add natural motion, atmospheric particles, gentle camera movement, and matching ambient mood. Avoid text artifacts, distorted hands, unstable faces, sudden cuts, unwanted objects, identity drift, and style changes.

## Rule
Choose the simplest engine that can achieve the desired emotional and visual result. Do not use stronger motion tools when subtle animation is enough.
