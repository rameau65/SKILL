# MUSIC_ENGINE

## Purpose
Create music and sound direction for videos, brands, meditation, documentaries, and channel identity.

## Tool
Suno and other AI music tools.

## Default Conditions
- Instrumental
- No vocals
- No drums
- Ambient
- Melodic
- Loop-friendly
- Minor key when emotional or meditative

## Output
1. 30-second signal music
2. 1-minute version
3. 3-minute version
4. 5-minute version
5. Advertising version
6. Meditation version
7. Darkwave / A24 version

## Base Prompt
Instrumental, no vocals, no drums, minor key, cinematic ambient electronic, dreamy melodic synth, soft analog texture, slow evolving pads, subtle pulse without percussion, Korean dawn mood, A24 opening atmosphere, clean mix, loop-friendly ending.

## Rule
Music should support image and emotion without overpowering narration or meditation.
