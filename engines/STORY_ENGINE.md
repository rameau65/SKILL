# STORY_ENGINE

## Purpose
Convert a topic into a clear narrative structure.

## Input
- Topic
- Audience
- Purpose
- Duration
- Tone

## Output
- Core message
- Logline
- 1/3/5 minute structure
- Scene list
- Narration
- Storyboard table
- Visual prompt seed

## Default Table

| Cut | Screen | Narration | Camera | Visual Prompt | Video Prompt |
|---|---|---|---|---|---|

## Rule
Every story must have: beginning, shift, realization, and return.
