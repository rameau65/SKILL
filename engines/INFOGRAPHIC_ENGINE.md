# INFOGRAPHIC_ENGINE

## Purpose
Transform complex information into clear visual structures, PPT pages, DOCX reports, and educational diagrams.

## Input
- Raw text
- Report
- Lecture material
- Proposal
- Story structure

## Output
| Page | Title | Core Message | Visual Element | Copy |
|---|---|---|---|---|

## Visualization Types
- Card layout
- Timeline
- Comparison table
- Flowchart
- Matrix
- Before/after
- Process diagram

## Principles
- One page, one message
- Short titles
- Explain simply
- Use visual hierarchy
- Reduce cognitive load

## Rule
If the content is complex, first reorganize it into 3–7 blocks before designing.
