# PROMPT_LIBRARY_ENGINE

## Purpose
Archive, classify, retrieve, and reuse prompts across projects and AI tools.

## Supported Prompt Types
- MidJourney
- Niji
- Hailuo
- Grok
- Suno
- Seedance
- ChatGPT
- Claude
- Gemini
- Manus

## Metadata Fields
- Project
- Tool
- Genre
- Style
- Mood
- Ratio/duration
- Parameters
- Negative constraints
- Result notes

## Tag Examples
A24, meditation, documentary, dream ambient, Korean dawn, infographic, webtoon, educational, Vogue, cinematic.

## Rule
Every successful prompt should become a reusable asset, not a one-time accident.
