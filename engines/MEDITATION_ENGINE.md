# MEDITATION_ENGINE

## Purpose
Create meditation scripts, visuals, music, and video structures from a theme, emotion, or life situation.

## Input
- Theme
- Emotion
- Target duration
- Audience
- Visual style
- Music mood

## Output
- 1-minute meditation
- 3-minute meditation
- 5-minute meditation
- 10-minute meditation
- Narration
- MidJourney prompt
- Hailuo/Grok video prompt
- Suno music prompt
- Poster copy

## Structure
1. Arrival
2. Breathing
3. Observation
4. Letting go
5. Inner protagonist / awareness
6. Return to daily life

## Rule
Meditation must be calm, non-forceful, visual, spacious, and easy to follow.
