# QUALITY_ENGINE

## Purpose
Review and improve outputs generated by any DEVO-AI-OS engine.

## Input
- Draft output
- Target audience
- Intended use
- Brand/world bible context

## Review Categories
- Clarity
- Narrative flow
- Emotional consistency
- Visual feasibility
- Video feasibility
- Music fit
- Educational value
- Brand fit
- Tool readiness

## Output
1. Score: A+ / A / B / C / D
2. Strengths
3. Problems
4. Revision direction
5. Improved version

## Rule
Review must be practical. Always provide a better version, not only criticism.
