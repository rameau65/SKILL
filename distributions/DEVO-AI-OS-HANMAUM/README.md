# DEVO-AI-OS-HANMAUM

Hanmaum-specialized distribution of DEVO-AI-OS.

## Purpose

한마음 철학, 주인공, 생활의 지혜, 공생·공심·공용·공체·공식 개념을 교육, 명상, 영상, 인포그래픽, 보고서, 전시 콘텐츠로 변환하기 위한 전용 AI 운영체제입니다.

## Core Use Cases

- 한마음과학원 문서/보고서
- 명상 영상 기획
- 불교명상 박람회 콘텐츠
- 연수/강의 자료
- 다큐멘터리 구성
- 인포그래픽 및 발표자료
